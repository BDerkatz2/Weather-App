# Weather Pulse

Neon-styled weather app with responsive layout, loading and error states, and clean React components.

## Setup

Create a `.env` file in the project root and add your OpenWeather API key:

```
VITE_OPENWEATHER_API_KEY=your_key_here
```

Install dependencies and run the dev server:

```
npm install
npm run dev
```

## Features

- City search and geolocation lookup
- Responsive layout with neon visual direction
- Loading skeletons and error messaging
- Auto-detected units with manual toggle
